# 绘本精读方法论详解

本文件是 skill 的深度参考。SKILL.md 给出工作流，本文件给出每个方法论的完整机理。**按需读取**。

## 一、对话式阅读 Dialogic Reading（Whitehurst, 1988）

迄今**唯一有大量 RCT 证据支持**的亲子共读方法，显著提升 2–6 岁儿童的表达性词汇与叙事能力。

### PEER 序列（每一页的互动骨架）

- **P**rompt — 引导孩子说关于书的内容
- **E**valuate — 对孩子的回应给予反馈（肯定、澄清、纠正）
- **E**xpand — 在孩子的回应基础上**加一句信息**（词汇、因果、情绪标签）
- **R**epeat — 让孩子把扩展后的句子再说一遍

举例：
> 父母（P）：你看这只小熊在做什么？
> 孩子：哭。
> 父母（E）：对，他在哭。
> 父母（E）：他是因为找不到妈妈所以难过得哭了。
> 父母（R）：你再说一次，他为什么哭？

### CROWD 五类提问（P 的具体形式）

| 类型 | 含义 | 例子 |
|------|------|------|
| **C**ompletion | 填空式 | "大灰狼吹啊吹，把房子都……" |
| **R**ecall | 回忆 | "刚才小猪怎么逃走的？" |
| **O**pen-ended | 开放式 | "你觉得这一页发生了什么？" |
| **W**h- questions | 五何 | 谁 / 什么 / 哪里 / 为什么 / 怎么 |
| **D**istancing | 联系现实 | "你有没有遇到过类似的事？" |

**搭配原则**：
- 1–3 岁：C + W 为主（最低语言门槛）
- 3–5 岁：CROWD 全用，但 C > O
- 5 岁+：O + D 为主（推动深度思考）

## 二、三读法（分层深入）

一本绘本应被读 5–10 次。三读法把重读组织成三个不同的工作模式：

### 第一读 · 图画漫步（Picture Walk）
只看图，**不读字**。
- 让孩子描述封面，预测故事
- 一页一页翻，问"这里发生了什么？"
- 目标：建立对图像叙事的整体直觉

### 第二读 · 文本细读
正常朗读 + PEER/CROWD 互动。
- 关注情节、因果、人物动机
- 父母不要全程提问，保持"朗读 3 页、停一次"的节奏

### 第三读 · 主题延展
不再读原文，只谈这本书。
- 用 Chambers 四问
- 讨论主题、作者意图、延伸到孩子的生活

## 三、Aidan Chambers"告诉我"四问

英国儿童文学权威的经典框架，比 CROWD 更适合 5 岁以上：

1. **喜欢什么？**（What did you like?）
2. **不喜欢什么？**（What didn't you like?）
3. **困惑什么？**（What puzzled you?）—— **最重要**
4. **发现了什么规律 / 联系？**（What patterns did you notice?）

"困惑"这一问是核心：它承认孩子有读不懂的地方，并鼓励把读不懂说出来，这比任何"你学到了什么"都更能推动深度阅读。

## 四、视觉素养 Visual Literacy

绘本是复合艺术。精读必须读图：

| 维度 | 观察点 |
|------|--------|
| **构图** | 谁在画面中心？谁在边缘？留白在哪边？ |
| **色彩** | 色调变化是否对应情绪变化？冷色块出现在哪一页？ |
| **视角** | 俯视 / 仰视 / 平视 传达了什么权力关系？ |
| **留白与跨页** | 翻页前后的构图变化藏着什么？跨页是为了什么？ |
| **图文反讽** | 图和文字说的是同一件事吗？ |

**经典反讽案例**：《母鸡萝丝去散步》全文只描述母鸡散步的路线，从不提狐狸，而每一幅图里狐狸都在遭殃。这是图文对位的顶级范例。

## 五、三种连接 Three Connections（Harvey & Goudvis）

训练孩子把书与世界联系起来，是"读出书本以外"的能力：

- **Text-to-Self 书 ↔ 我**："这让你想到自己的什么经历？"
- **Text-to-Text 书 ↔ 书**："这和我们之前读的 X 有什么像/不像？"
- **Text-to-World 书 ↔ 世界**："真实世界里有这种事吗？"

## 六、Bloom 分层提问（由浅入深）

| 层级 | 动词 | 问题模板 |
|------|------|---------|
| 记忆 | 指认、复述 | "小兔子叫什么？" |
| 理解 | 转述、归纳 | "用你的话讲讲刚才发生了什么" |
| 应用 | 迁移 | "如果你是他，你会怎么做？" |
| 分析 | 拆解、比较 | "为什么作者要把这一页画成黑色？" |
| 评价 | 评判 | "你觉得他这样做对吗？为什么？" |
| 创造 | 构造、改编 | "我们一起编一个不一样的结局好吗？" |

**常见错误**：父母 80% 的提问集中在"记忆"层。精读要刻意上移到分析 / 评价 / 创造。
