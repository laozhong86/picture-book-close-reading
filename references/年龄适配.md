# 年龄适配矩阵

精读必须"适龄适孩"。SKILL.md 调用此文件来决定：**提问风格、单次时长、重心**。

## 主矩阵

| 年龄段 | 阅读重心 | 单次时长 | 首选技巧 | 次选技巧 |
|--------|---------|---------|---------|---------|
| 0–18 个月 | 指认 / 拟声 / 韵律 / 感官触觉 | 3–8 分钟 | 手指点读、夸张语调、指物命名 | 触摸书、翻页本能 |
| 18 个月–3 岁 | 词汇 / 重复结构 / 情绪命名 | 8–15 分钟 | CROWD 中的 **C 填空** 和 **W 五何** | 预测："你猜下一页……" |
| 3–5 岁 | 因果 / 角色动机 / 情绪细分 | 15–20 分钟 | **PEER 全序列**；CROWD 均衡使用 | 三读法：第一读图画漫步可正式上线 |
| 5–7 岁 | 主题 / 视觉素养 / 反讽 | 20–30 分钟 | **Chambers 四问**；Bloom 分析/评价层 | 三种连接，尤其 Text-to-Text |
| 7 岁+ | 文学技巧 / 作者意图 / 创作迁移 | 25–40 分钟 | Bloom 创造层 / 写作仿写 | 跨书主题比较、经典同题对读 |

## 性格 × 提问风格

| 性格 | 优先用 | 避免用 |
|------|-------|-------|
| 害羞 / 话少 | Completion 填空、选择题、指认 | Open-ended 长开放题 |
| 外向 / 话痨 | Open-ended 开放题、Distancing 联结 | 太多 C 填空（会被嫌无聊） |
| 敏感 | 情绪命名 + 三种连接中的 Text-to-Self | 强度高的评价题 |
| 好动 | 短提问 + 立刻起身做延伸活动 | 连续坐读 30 分钟 |
| 爱提问 | 反问、Chambers 的"困惑"问 | 父母自答 |
| 完美主义 | 允许"没有标准答案"的创造题 | 任何"对 / 错" 判定 |

## 兴趣适配（联结弹药）

痴迷主题在"5 个关键提问"的**联结层**必须被调用。示例映射：

| 孩子痴迷 | 可迁移问法 |
|---------|-----------|
| 恐龙 | "如果这一页的主角换成霸王龙会怎么样？" |
| 汽车 | "谁来开车？车上装什么？用什么动力？" |
| 公主 / 魔法 | "如果她有魔法，她会怎么解决？" |
| 太空 | "这个故事搬到火星上，会哪里不一样？" |
| 昆虫 | "用蚂蚁的角度看这一页，会看到什么？" |
| 厨房 / 吃 | "他们今天会吃什么？谁做饭？" |

这类迁移看似"不务正业"，实际上是最有效的"主动注意力抓手"。

## 父母期待 × 输出权重

| 期待 | A4 权重调整 |
|------|-----------|
| 词汇量 | 第二读 PEER 的 **E（扩展）**要明确给出扩展词汇示例 |
| 情绪认知 | 5 问中至少 2 问涉及情绪命名；提供"情绪词库"候选 3–5 个 |
| 想象力 | 5 问中必含创造层 + 延伸活动至少 1 个是开放式 |
| 专注力 | 第二读时提醒"朗读 3 页、停一次"，避免频繁打断 |
| 亲子连接 | Distancing 联结题数量上调；延伸活动设计为共做型 |

## 单次时长安全上限

> **超过孩子年龄对应时长还没结束，就该停止**。精读不是越长越好。分两三次完成优于一口气读完再无复读。
